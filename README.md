
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Zer0Power/Steam-Comment-Bot
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Edit Config File
   ```json
         "username":"username",
        "password":"password",
        "admin":"adminID64",
        "identitySecret":"Search On Google"
    ```
4. Run the Project
   ```sh
   npm run test
   ```
